**SQL 검증**:

### **TC-ADMIN-102: 문제 생성 - 정상 케이스**
**우선순위**: P1  
**목적**: 새 문제 생성 기능 정상 작동 확인

**전제조건**:
- 관리자로 로그인
- 학습 세트 존재 (study_set_id = 1)

**테스트 단계**:
1. `/admin/study_sets/1/questions/new` 접속
2. 폼 필드 입력:
   - **문제 내용**: "사회복지의 정의는?"
   - **문제 유형**: "multiple_choice"
   - **난이도**: 3
   - **주제**: "사회복지개론"
   - **설명**: "사회복지의 기본 개념"
   - **옵션 4개**:
     - 옵션 1: "개인의 복지 증진" (정답 ✓)
     - 옵션 2: "국가의 경제 발전"
     - 옵션 3: "기업의 이윤 추구"
     - 옵션 4: "정치적 안정"
3. "저장" 버튼 클릭
4. 리다이렉트 확인: `/admin/study_sets/1`
5. 성공 메시지 확인: "문제가 성공적으로 생성되었습니다."

**예상 결과**:
- ✅ 문제 DB에 저장됨
- ✅ 옵션 4개 저장됨
- ✅ 정답 옵션 `is_correct = true`
- ✅ 성공 메시지 표시
- ✅ 학습 세트 페이지로 리다이렉트

**SQL 검증**:
```sql
-- 문제 생성 확인
SELECT * FROM questions WHERE content = '사회복지의 정의는?' ORDER BY id DESC LIMIT 1;

-- 옵션 생성 확인
SELECT * FROM options WHERE question_id = <new_question_id>;

-- 정답 확인
SELECT content, is_correct FROM options WHERE question_id = <new_question_id> AND is_correct = true;
```

## 📦 **P2: CSV 일괄 가져오기 테스트**

### **TC-ADMIN-201: CSV 일괄 가져오기 - 정상 케이스**
**우선순위**: P2  
**목적**: CSV 파일로 문제 일괄 추가 기능 확인

**전제조건**:
- 관리자로 로그인
- 유효한 CSV 파일 준비:

```csv
study_set_id,content,question_type,difficulty,topic,options,correct_answer,explanation
1,"사회복지의 목표는?",multiple_choice,3,"사회복지개론","개인 복지|사회 안정|경제 발전|정치 안정",0,"사회복지는 개인의 복지 증진을 목표로 합니다."
1,"사회복지사의 역할은?",multiple_choice,2,"사회복지실천","상담|치료|옹호|모두 해당",3,"사회복지사는 다양한 역할을 수행합니다."
```

**테스트 단계**:
1. `/admin/questions` 접속
2. "CSV 가져오기" 버튼 클릭 (super_admin만 보임)
3. CSV 파일 선택
4. "가져오기" 버튼 클릭
5. 성공 메시지 확인: "2개의 문제가 성공적으로 추가되었습니다."
6. 문제 목록에서 새 문제 확인

**예상 결과**:
- ✅ 2개 문제 DB에 추가됨
- ✅ 각 문제마다 옵션 4개 생성됨
- ✅ 정답 옵션 `is_correct = true`
- ✅ 성공 메시지 표시

**SQL 검증**:
```sql
-- 새로 추가된 문제 확인
SELECT * FROM questions WHERE content IN ('사회복지의 목표는?', '사회복지사의 역할은?');

-- 옵션 확인
SELECT q.content, o.content, o.is_correct 
FROM questions q 
JOIN options o ON q.id = o.question_id 
WHERE q.content = '사회복지의 목표는?';
```
