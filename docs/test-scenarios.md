# ExamsGraph 종합 테스트 시나리오

**작성일**: 2026-01-18  
**목적**: UI는 구현되어 있지만 작동하지 않거나 에러가 있는 기능 발견 및 수정

## 🎯 **테스트 전략**

### **1. 핵심 사용자 플로우 테스트**
- 회원가입 → 로그인 → 학습세트 생성 → PDF 업로드 → 문제 풀이 → 결과 확인

### **2. 각 페이지별 기능 테스트**
- 모든 버튼, 링크, 폼 동작 확인
- 에러 메시지 확인
- 데이터 표시 확인

### **3. 엣지 케이스 테스트**
- 빈 데이터 상태
- 잘못된 입력
- 권한 없는 접근

## 🐛 **예상 버그 목록**

### **High Priority**
1. **Practice Mode / Exam 시작 에러**
   - 증상: "RecordNotFound" 에러
   - 위치: `/study_sets/:id` → "Practice Mode" 또는 "Start Exam" 클릭
   - 원인: 세션 생성 로직 또는 권한 검증 문제

2. **Questions 라우트 에러**
   - 증상: `/study_sets/:id/questions` 접근 시 "Study material not found"
   - 위치: Questions 컨트롤러
   - 원인: `study_set_id` vs `study_material_id` 혼동

3. **자격증 선택 동적 로드 미작동**
   - 증상: 카테고리 선택 시 자격증 목록 업데이트 안 됨
   - 위치: `/study_sets/new`
   - 원인: Stimulus 컨트롤러 미구현

### **Medium Priority**
4. **PDF 처리 상태 업데이트 안 됨**
   - 증상: 처리 완료 후에도 "처리 중" 상태
   - 위치: Study materials 목록
   - 원인: `ProcessPdfJob`에서 `status` 업데이트 누락

5. **Pagination 링크 에러**
   - 증상: "다음" 버튼 클릭 시 404
   - 위치: Questions 목록
   - 원인: 잘못된 라우트 생성

### **Low Priority**
6. **빈 데이터 상태 UI**
   - 증상: 데이터 없을 때 빈 화면
   - 위치: 여러 페이지
   - 원인: 빈 상태 메시지 미구현
