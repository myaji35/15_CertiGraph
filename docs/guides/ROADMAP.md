# ExamsGraph 개발 로드맵

## 📋 운영 시나리오

1. **회원가입/로그인** → 2. **시험일자 선택** → 3. **1만원 결제** → 4. **PDF 업로드** → 5. **문제 학습** → 6. **답안 입력** → 7. **반복 학습/체점** → 8. **취약점 분석** → 9. **취약 문제 집중 학습** → 10. **예상 점수 평가**

## 🎯 개발 단계

### Phase 1: MVP (현재 완료)
- ✅ 기본 UI/UX
- ✅ PDF 파싱 및 문제 표시
- ✅ 학습 세션 관리
- ✅ 문제 네비게이션
- ✅ Clerk 인증 연동

### Phase 2: 핵심 기능 (1주차)
- [ ] **결제 시스템**
  - 토스페이먼츠/포트원 연동
  - 시험일자별 1만원 구독 모델
  - 결제 내역 관리

- [ ] **답안 관리 시스템**
  - 관리자 답안 입력 인터페이스
  - 답안 크라우드소싱 (사용자 제보)
  - 답안 검증 시스템 (3명 이상 동일 답안)

### Phase 3: 학습 고도화 (2주차)
- [ ] **AI 취약점 분석**
  - GPT-4 기반 오답 패턴 분석
  - 개념별 취약도 매핑
  - 맞춤형 학습 경로 생성

- [ ] **취약 문제 집중 모드**
  - 틀린 문제만 모아서 풀기
  - 보기 섞기 활성화 (정답 암기 방지)
  - 유사 문제 추천

### Phase 4: 평가 시스템 (3주차)
- [ ] **예상 점수 예측**
  - 과목별 점수 예측
  - 과락 위험도 알림
  - 학습 진도율 기반 합격 확률

- [ ] **모의고사 모드**
  - 실전과 동일한 시간 제한
  - 전체 75문제 풀기
  - 즉시 채점 및 해설

### Phase 5: 커뮤니티 & 부가기능 (4주차)
- [ ] **학습 커뮤니티**
  - 문제별 토론 게시판
  - 오답노트 공유
  - 스터디 그룹 매칭

- [ ] **성과 보장 프로그램**
  - 불합격 시 다음 시험 무료
  - 학습 완료율 90% 이상 조건

## 🏗️ 기술 스택 추가

### 결제
- **Toss Payments** or **PortOne** (아임포트)
- Webhook을 통한 결제 검증

### 데이터 분석
- **Python FastAPI** + **Pandas** (데이터 분석)
- **Scikit-learn** (예측 모델)

### 실시간 기능
- **Supabase Realtime** (답안 크라우드소싱)
- **WebSocket** (실시간 채점)
