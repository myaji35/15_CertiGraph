# BMad Test Scenarios for CertiGraph

## 테스트 시나리오 개요
CertiGraph (AI 자격증 마스터) 플랫폼의 주요 기능을 검증하는 E2E 테스트 시나리오

## 1. 인증 및 권한 (Authentication & Authorization)

### 1.1 회원가입 프로세스
- **시나리오**: 신규 사용자가 회원가입을 완료한다
- **검증 항목**:
  - 이메일 유효성 검증
  - 비밀번호 복잡도 체크
  - 중복 이메일 거부
  - 회원가입 완료 후 자동 로그인

### 1.2 로그인/로그아웃
- **시나리오**: 기존 사용자가 로그인하고 로그아웃한다
- **검증 항목**:
  - 유효한 credentials로 로그인 성공
  - 잘못된 credentials로 로그인 실패
  - 로그아웃 후 protected route 접근 차단
  - Remember me 기능

### 1.3 소셜 로그인
- **시나리오**: Google OAuth로 로그인한다
- **검증 항목**:
  - Google 로그인 버튼 표시
  - OAuth 리다이렉트 처리
  - 로그인 후 사용자 정보 표시

## 2. 학습 자료 관리 (Study Materials)

### 2.1 PDF 업로드 및 처리
- **시나리오**: 사용자가 시험 문제 PDF를 업로드한다
- **검증 항목**:
  - PDF 파일 업로드 UI 표시
  - 업로드 진행 상태 표시
  - Upstage OCR 처리 상태
  - 처리 실패 시 재시도 버튼 표시
  - 처리 완료 후 자료 목록에 추가

### 2.2 학습 자료 목록 조회
- **시나리오**: 업로드된 학습 자료 목록을 확인한다
- **검증 항목**:
  - 자료 목록 페이지네이션
  - 자료 검색 기능
  - 자료 상태 표시 (처리중/완료/실패)
  - 자료 상세 정보 표시

### 2.3 학습 자료 삭제
- **시나리오**: 불필요한 학습 자료를 삭제한다
- **검증 항목**:
  - 삭제 확인 다이얼로그
  - 삭제 후 목록 업데이트
  - 연관 데이터 정리 확인

## 3. 모의시험 (Mock Exam)

### 3.1 시험 시작
- **시나리오**: 사용자가 모의시험을 시작한다
- **검증 항목**:
  - 시험 세트 선택
  - 시험 설정 (시간제한, 문제 수)
  - 시험 시작 확인

### 3.2 시험 진행
- **시나리오**: 사용자가 문제를 풀고 답안을 제출한다
- **검증 항목**:
  - 문제 표시 및 네비게이션
  - 답안 선택 및 수정
  - 북마크 기능
  - 타이머 표시
  - 자동 저장

### 3.3 시험 결과
- **시나리오**: 시험 완료 후 결과를 확인한다
- **검증 항목**:
  - 점수 및 통계 표시
  - 오답 노트 생성
  - 문제별 해설 표시
  - 결과 저장
